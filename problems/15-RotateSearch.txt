Problem Name: HELP Hamza TO SEARCH 
Problem Description:
Hamza had a sorted array of numbers from which he had to find a given number quickly. His friend by mistake rotated the array. Now Rahul doesn't have time to sort the elements again. Help him to quickly find the given number from the rotated array.  

Hint - Think Binary Search!

Input Format: The first line contains N - the size of the array. the next N lines contain the numbers of the array.The next line contains the item to be searched.
Sample Input: 5
4
5
1
2
3
2
Output Format: Output the index of number in the array to be searched. Output -1 if the number is not found.
Sample Output: 3
