Problem Name: Maximum Subarray Sum (Divide and Conquer)
Problem Constraints: 1 <= N <= 100000
1 <= t <= 20
0 <= A[i] <= 100000000
Problem Description:
You are given a one dimensional array that may contain both positive and negative integers, find the sum of contiguous subarray of numbers which has the largest sum.
For example, if the given array is {-2, -5, 6, -2, -3, 1, 5, -6}, then the maximum subarray sum is 7.
**NOTE**:- please solve this problem using Divide and Conquer instead of kadane's algorithm.

Input Format: The first line consists of number of test cases T.v Each test case consists of N followed by N integers.v Sample Input: 2
4
1 2 3 4
3
-10 5 10
Output Format: The maximum subarray sum
Sample Output: 10
15
